Movie Search - description

A detailed analysis of the project

About

Live View on Github

This is an application built using React.js library and its purpose was to allow the user to look for movies within a database by enetering the name (fully or partially) of a movie of choice as input and pressing the submit button afterwords.

After pressing the button, the name of the movie will be sent to the OMDb API via URL and if the movie is found inside the OMDb database, it will return a JSON file containing information about the search result including the year of release and the URL for a poster (if there is one available). For each result found in the array of objects returned by the OMDb API there will be a React component that displays the information related to that result.

The main component (App.js) will show a different result depending on the state of 3 variables. If a poster is not available for a movie, a default poster will be loaded.

Side note

The motivation behind this project was my interest towards understanding important React features such as hooks ("useState", "useEffect", "useReducer") and the exchange of data between React components that have diferent relationships with each other inside the hierarchy with the help of props .
